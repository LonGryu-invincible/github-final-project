# Simple Interest Calculator

This repository contains a simple Bash script to calculate simple interest based on user input.

## Formula

Simple Interest is calculated using the formula:

```text
Simple Interest = (Principal × Rate × Time) / 100
```

Where:

- **Principal** is the original amount of money.
- **Rate** is the annual rate of interest.
- **Time** is the time period in years.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/simple-interest-calculator.git
```

2. Go to the project folder:

```bash
cd simple-interest-calculator
```

3. Give execute permission to the script:

```bash
chmod +x simple-interest.sh
```

4. Run the script:

```bash
./simple-interest.sh
```

## Example

```text
Simple Interest Calculator
Enter the principal amount: 1000
Enter the rate of interest: 5
Enter the time period in years: 2

```
